Descripción del Producto

El Switch Administrable TP-Link TL-SG3424P es una solución de red avanzada que combina 24 puertos Gigabit PoE con 4 puertos SFP, ofreciendo un rendimiento excepcional para entornos empresariales que requieren alta disponibilidad y gestión avanzada de red.

Características Principales

  • 24 puertos Gigabit PoE+ con un presupuesto total de 320W
  • 4 puertos SFP para conexiones de fibra óptica
  • Funciones avanzadas de capa L2+
  • Interfaz de gestión web intuitiva
  • Diseño robusto para montaje en rack

Ideal para empresas medianas y grandes que requieren una solución de red escalable, segura y de alto rendimiento para soportar aplicaciones críticas de negocio, videovigilancia IP y telefonía IP.

Especificaciones Técnicas

Modelo: TL-SG3424P
Puertos: 24 puertos 10/100/1000Mbps RJ45 con PoE+ (802.3at/af), 4 puertos SFP
Capacidad de Conmutación: 128 Gbps
Ancho de Banda por Puerto: 10/100/1000Mbps (RJ45), 1000Mbps (SFP)
Memoria: 512MB SDRAM, 128MB Flash
Jumbo Frame: Hasta 10KB
Tabla de Direcciones MAC: 16K
VLANs: Hasta 4K VLANs activas (IEEE 802.1Q)
Protocolos: IEEE 802.1D, IEEE 802.1p, IEEE 802.1Q, IEEE 802.1s, IEEE 802.1w, IEEE 802.1x, IEEE 802.3ab, IEEE 802.3ad, IEEE 802.3i, IEEE 802.3u, IEEE 802.3x, IEEE 802.3z

Características Principales

Conectividad y Rendimiento

  • 24 puertos Gigabit Ethernet con soporte PoE+ (802.3at/af)
  • 4 puertos SFP para conexiones de fibra óptica
  • Capacidad de conmutación de 128 Gbps
  • Ancho de banda no bloqueante de 64 Gbps
  • Rendimiento de reenvío de 95.2 Mpps

Seguridad Avanzada

  • Autenticación 802.1X, autenticación basada en puerto/mac
  • Listas de control de acceso (ACL) para mayor seguridad
  • Defensa contra ataques DoS
  • Control de tormentas de difusión/unión múltiple/unicast
  • Soporte para RADIUS/TACACS+
  • VLAN privada (PVLAN)

Gestión y Monitoreo

  • Interfaz web intuitiva para configuración fácil
  • CLI para administradores avanzados
  • Soporte para SNMP v1/v2c/v3
  • Monitoreo de puertos y tráfico en tiempo real
  • Actualizaciones de firmware remotas
  • Soporte para Syslog

Calidad de Servicio (QoS)

  • Clasificación de tráfico basada en puerto, 802.1p, DSCP
  • 4 colas por puerto
  • Priorización de tráfico para voz y video
  • Control de ancho de banda por puerto
